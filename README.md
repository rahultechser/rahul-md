# 🤖 Rahul-MD WhatsApp Bot

A powerful and multi-functional WhatsApp bot built with Node.js and Baileys.

## 🚀 Features
- **AI Chat:** Chat with Rahul-MD AI.
- **Downloader:** Download videos and songs easily.
- **Sticker Maker:** Convert images to stickers.
- **Lightweight:** Fast and easy to deploy.

## 🛠 Setup
1. Clone this repo.
2. Run `npm install`.
3. Start with `npm start`.
4. Scan the QR code.
